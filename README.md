# Personal Finance App

Repository layout:

- `frontend/` contains the Next.js frontend app.
- `backend/` contains the Fastify API.

Run frontend commands from `frontend/`.
Run backend commands from `backend/`.

For the OpenAI-powered assistant, set `OPENAI_API_KEY` in the backend environment.
Optional: set `OPENAI_MODEL` to override the default model (`gpt-5.4-mini`).
